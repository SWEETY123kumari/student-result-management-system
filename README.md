# 🎓 Student Result Management System

A dynamic web application designed to digitize student records, allowing for secure admin management and instant online result retrieval. Built to streamline the academic workflow for schools and colleges.

---

## Key Features

### For Students
* **Instant Result Access:** Retrieve marks securely by entering the Roll Number.
* **Detailed Performance View:** Clear breakdown of subject-wise marks (Maths, Science, English, Hindi, Sst) and total percentage.
* **Print-Ready Marksheets:** Built-in functionality to print or save marksheets as PDF.

### For Admins
* **Secure Session Management:** Robust admin authentication using PHP sessions to restrict unauthorized access.
* **Efficient CRUD Operations:** Seamlessly add, update, and manage student data through a centralized dashboard.
* **Modular Database Handling:** Streamlined connectivity with a dedicated database handler.

---

## Tech Stack
* **Frontend:** HTML5, CSS3, Bootstrap
* **Backend:** PHP
* **Database:** MySQL (XAMPP Server)

## ⚙️ How to Run Locally

1. **Setup Environment:**
   * Install and open **XAMPP**.
   * Start the **Apache** and **MySQL** modules from the XAMPP Control Panel.

2. **Database Setup:**
   * Open [http://localhost/phpmyadmin/](http://localhost/phpmyadmin/) in your browser.
   * Create a new database and import the `srms.sql` file provided in the repository.

3. **Project Configuration:**
   * Move the project folder to your `htdocs` directory (e.g., `C:\xampp\htdocs\`).
   * Update the credentials in `db.php` if your database settings differ from the default local configuration.

4. **Access the App:**
   * Open your browser and visit: `http://localhost/student-result-management-system/`
